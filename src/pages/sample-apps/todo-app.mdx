This is a simple GraphQL application to demonstrate Dgraph's GraphQL capabilities.
There are three versions of the application which are built using ReactJS, Vue.js,
and Flutter. All of them powered by the GraphQL API of Dgraph.

### Functionalites

- Add a new task to the list
- Display all tasks from the list
- Edit an existing task
- Delete an existing task
- Toggle the completed status of the tasks
- Clear completed tasks
