How to use the GraphQL API. 

Dgraph serves [spec compliant
GraphQL](https://graphql.github.io/graphql-spec/June2018/) over HTTP. By
default, GraphQL is served at `http://localhost:8080/graphql`. Both GET and POST
requests are served.
