## Dgraph

**Dgraph** is an open-source, distributed, and transactional graph database with a native
GraphQL interface. Using Dgraph, you can get a GraphQL API without writing any code.
You need to define just the GraphQL type definitions in a schema file, Dgraph
auto-generates the mutations and queries in the GraphQL API for you.

This GraphQL API provide you with capabilities to perform CRUD operations on the types defined.

Dgraph provides custom directives for your GraphQL type definitions. You need to add them to
your GraphQL type definitions be able to make use of some of Dgraph's advanced database features.
